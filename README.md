<h1 align="center">vue-sun-ui</h1>

## Install

```bash
# Using npm
npm i vue-sun-ui -S

# Using yarn
yarn add vue-sun-ui
```

## Quickstart

```js
import Vue from 'vue'
// 引入vue-sun-ui
import SunUi from 'vue-sun-ui'
// 导入vue-sun-ui样式
import 'vue-sun-ui/dist/vue-sun-ui.css'

Vue.use(SunUi)
```
